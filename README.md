# 오늘 뭐 먹지 Agent

재료, 컨디션, 환경을 입력하면 오늘 먹기 좋은 메뉴를 추천해주는 초보자용 Python 에이전트입니다.

기본 추천 기능은 외부 패키지 없이 실행됩니다.

## 실행 방법

터미널에서 아래 명령어를 실행하세요.

```bash
cd meal_agent
python3 main.py
```

브라우저 화면으로 쓰고 싶으면 아래 명령어를 실행하세요.

```bash
python3 web_app.py
```

터미널에 표시되는 `http://127.0.0.1:8000` 같은 주소를 브라우저에서 열면 됩니다. 8000번 포트가 사용 중이면 자동으로 다음 포트를 사용합니다.

## Vercel 배포

이 프로젝트는 Vercel 배포용 구조도 포함합니다.

- `vercel.json`: 첫 화면과 정적 파일 경로를 연결합니다.
- `api/recommend.py`: Vercel 서버리스 함수로 추천 API를 실행합니다.
- `main.py`: 로컬 앱과 Vercel API가 함께 쓰는 추천 로직입니다.

Vercel에서 프로젝트를 만들 때 루트 디렉터리는 `meal_agent` 폴더로 두면 됩니다.

CLI로 배포한다면 `meal_agent` 폴더에서 아래 명령어를 실행합니다.

```bash
cd meal_agent
vercel
```

배포가 끝나면 `https://...vercel.app` 주소가 생기고, 그 링크를 친구에게 공유할 수 있습니다.

## 첫 버전에서 하는 일

- 가지고 있는 재료를 입력받습니다.
- 현재 컨디션을 입력받습니다.
- 날씨, 시간대, 설거지 여부 같은 환경을 입력받습니다.
- 조리 가능 시간과 조리도구를 반영합니다.
- 메뉴 후보를 점수화해서 TOP 3를 추천합니다.
- 가장 잘 맞는 메뉴의 간단 레시피를 보여줍니다.
- 알레르기나 싫어하는 재료를 저장할 수 있습니다.

## 추천 입력 예시

```text
가지고 있는 재료: 계란, 두부, 김치, 밥, 양파
지금 컨디션: 피곤하고 속이 더부룩함
환경: 비 오는 저녁, 설거지 적게
몇 분 안에 먹고 싶나요?: 20
오늘 피하고 싶은 재료/알레르기:
사용 가능한 조리도구: 냄비, 프라이팬
매운맛은 어느 정도가 좋아요?: 약하게
```
